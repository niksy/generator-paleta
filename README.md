# generator-paleta

[![Build Status][ci-img]][ci]

[Yeoman][yeoman] generator for my projects.

> _paleta_ is Croatian word for _palette_, a thin board on which a painter lays
> and mixes colors.

## Install

```sh
npm install -g @niksy/generator-paleta
```

## Usage

```sh
yo @niksy/paleta
```
